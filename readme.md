# YuzuParse, an audio file parser & player

YuzuParse parses audio files into PCM streams and can play them with PortAudio.

## Parsable/Playable Formats

* __filetype / codec__
* [ ] .MP3 / MP3
* [x] .WAV / PCM `completed January 10th, 2026`
* [ ] .WAV / MP3
* [ ] .OGG / VORBIS
* [ ] .OGG / FLAC
* [ ] .OGG / OPUS
* [ ] .FLAC / FLAC
* [ ] .AIFF / PCM
* [ ] .AAC / AAC
* [ ] .M4A / AAC
